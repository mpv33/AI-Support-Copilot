# Chat guide

No need to read all support docs first — ask in plain language. The copilot searches articles and shows sources under each answer.

## Example questions (sidebar)

| Topic | Try asking |
| --- | --- |
| Refunds | Refund policy; refund after 20 days on annual plan |
| Access | AUTH_403 on dashboard; access after role change |
| Orders | Where is order 123? |
| Replies | Draft a refund policy reply |

**Tip:** Include plan type, error code, or order ID for better answers.
